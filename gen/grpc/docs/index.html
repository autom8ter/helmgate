Method NameRequest TypeResponse TypeDescription
CreateApp AppInput App

CreateApp creates a new stateless application(k8s deployment & service) within a single namespace(k8s namespace) the namespace will automatically be created if one does not already exist
UpdateApp AppInput App

UpdateApp edits/patches an existing stateless application(k8s deployment & service) within an existing namespace(k8s namespace)
DeleteApp Ref .google.protobuf.Empty

DeleteApp deletes a single stateless application(k8s deployment & service) within an existing namespace
GetApp Ref App

GetApp gets an app by name within an existing namespace
ListApps Namespace Apps

ListApps lists all apps within an existing namespace
ListNamespaces .google.protobuf.Empty Namespaces

ListNamespaces lists all namespaces created by kdeploy
DeleteAll Namespace .google.protobuf.Empty

DeleteAll deletes all apps/tasks within an existing namespace

DeleteTask Ref .google.protobuf.Empty

ListTasks Namespace Tasks

StreamLogs streams logs from an app/task within an existing namespace. Streams are opened to all replicas & converted into a single stream
